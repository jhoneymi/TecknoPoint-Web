Punto de Venta Web
Este es un proyecto de punto de venta web desarrollado en Python Flask. Proporciona una interfaz de usuario intuitiva para gestionar las ventas, inventario y clientes de un negocio.

Paso de Ejecución del Proyecto:

    1. Instalación de Dependencias: Asegúrate de tener instalado Python en tu sistema. Luego, puedes instalar las dependencias del proyecto utilizando pip:
        -pip install -r requirements.txt

    2. Configuración de la Base de Datos: Asegúrate de tener un servidor de base de datos MySQL disponible. En el archivo app.py, se utiliza la librería flask_mysqldb para la conexión a la base de datos MySQL. Asegúrate de actualizar las configuraciones de conexión con tu base de datos en la sección correspondiente del archivo config.txt.

    3. Ejecución del Servidor: Una vez configurada la base de datos, puedes ejecutar el servidor Flask con el siguiente comando:
        -python app.py

    4. Acceso a la Aplicación: Abre tu navegador web y visita la dirección http://localhost:5000 para acceder a la aplicación de punto de venta.

Librerías Utilizadas:

    1. Flask: Un microframework web para Python.
    2. Flask-SocketIO: Extensión de Flask para la comunicación en tiempo real basada en WebSockets.
    3. Flask-MySQLdb: Una extensión de Flask que proporciona soporte para la base de datos MySQL.
    4. Bcrypt: Una biblioteca de hash de contraseñas para Python.
    5. PDFKit: Una biblioteca para generar archivos PDF a partir de HTML en Python.
    6. Datetime: Un módulo de Python para manejar fechas y horas.
    7. IO: Un módulo de Python para trabajar con flujos de entrada y salida.
    8. Random: Un módulo de Python para generar números aleatorios.
    9. String: Un módulo de Python para trabajar con cadenas de texto.
    10. Smtplib: Un módulo de Python para enviar correos electrónicos.
    11. Email: Un módulo de Python para manejar mensajes de correo electrónico.
